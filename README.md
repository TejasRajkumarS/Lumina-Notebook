## 📖 About The Project

**Lumina-Notebook** is a smart application generated via Google AI Studio that allows you to integrate advanced language models seamlessly into a dynamic notebook environment. Built with modern web technologies, it leverages the power of the **Gemini API** for AI generation and **Supabase** for robust backend interactions.

View the original app configuration in AI Studio: [Lumina-Notebook Project](https://ai.studio/apps/6db355b2-3b64-4ae2-945b-1378ceabc54c)

## ✨ Features

- **⚡ Blazing Fast Frontend:** Powered by Vite and TypeScript for an optimized, type-safe development experience.
- **🧠 AI Integration:** Deeply integrated with Google's Gemini API to bring intelligent generation and assistance directly to the notebook.
- **🗄️ Scalable Backend:** Uses Supabase for data management and persistence.
- **🎨 Modern UI/UX:** Clean, responsive design implemented with standard web technologies.

## 🛠️ Tech Stack

- **Language:** [TypeScript](https://www.typescriptlang.org/) (97% of codebase)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Backend/Database:** [Supabase](https://supabase.com/)
- **AI/LLM:** [Google Gemini API](https://ai.google.dev/)
